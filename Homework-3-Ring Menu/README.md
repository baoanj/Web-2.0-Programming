# Web 2.0 Programming - Homework3: Ring Menu

_ @+环形菜单出现在页面下部，靠近底边，居中
_ 其关闭时（鼠标未指向），只显示缩小的@+按钮，效果如图：
_ 鼠标指向之后，显示出环形菜单，包括5个控制按钮和一个信息栏（info-bar）

>本次作业没有任何JavaScript，动画部分可用CSS3的transition来实现。
